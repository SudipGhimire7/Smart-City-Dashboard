# 🛠️ Dashboard Tech Stack & Tooling

This dashboard is built as a professional Smart City Command Center (**CityDesk**), combining a clean, high-fidelity user interface with real-time data monitoring systems.

### ⚛️ Core Framework
- **[React 18](https://react.dev/)**: For high-performance, component-based user interface management.
- **[Vite 6](https://vitejs.dev/)**: Lightning-fast build tool ensuring an optimized production bundle and rapid development.

### 🎨 Styling & Design
- **[Tailwind CSS v4](https://tailwindcss.com/)**: Powering a unified, professional design system with a focus on legibility and clean aesthetics.
- **[Motion](https://www.framer.com/motion/)**: Used for subtle micro-interactions and smooth layout transitions that enhance user experience without distraction.
- **Typography**: Optimized using the **Outfit** variable font for a premium, accessible reading experience.

### 🧱 Component Architecture
- **[Shadcn UI](https://ui.shadcn.com/)**: A robust collection of accessible UI components built with Radix UI primitives.
- **[Lucide React](https://lucide.dev/)**: A standard library of consistent, professional icons used throughout the city management system.
- **[Sonner](https://sonner.stevenly.me/)**: Streamlined toast notifications for high-priority operational alerts.

### 📊 Data & Systems
- **[Recharts](https://recharts.org/)**: Advanced charting used for visualizing the Sustainability Index and infrastructure analytics.
- **Simulation Engine**: React-driven state logic for monitoring real-time urban metrics including Traffic Density, Air Quality (AQI), and Grid Stability.

### 🤖 Development Partner
- **[Antigravity AI](https://github.com/google-deepmind/advanced-agentic-coding)**: AI pair-programmer used for architectural modernization, UI/UX refinement, and implementing detailed city-wide mapping systems.

---

## Architecture Overview
The dashboard follows a **Responsive Management Layout** that scales from ultra-wide desk monitors to standard handheld devices. The system is designed for high-availability monitoring, with integrated **Emergency Override** protocols that allow for instantaneous situational awareness and infrastructure priority shifts.
